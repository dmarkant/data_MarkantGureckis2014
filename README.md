This repository contains the raw data for: 

    Markant, D., & Gureckis, T. M. (2014). Is it better to select or to receive? Learning via active and passive hypothesis testing. Journal of Experimental Psychology: General, 143(1), 94–122. https://doi.org/10.1037/a0032108

There is separate file for each participant (located in data_raw, divided by task and training condition into separate folders). This file provides guidance about how these data files are structured. Please direct any additional questions to Doug Markant (dmarkant@charlotte.edu).

See additional documentation in Data structure.docx.
